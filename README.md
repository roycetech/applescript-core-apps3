# Overview

This repository was refactored from the original [applescript-core](https://github.com/roycetech/applescript-core) repository to be used for 3rd party apps and libraries.

It requires the [applescript-core](https://github.com/roycetech/applescript-core) repository to be installed first.

## Installation

```sh
curl -fsSL https://raw.githubusercontent.com/roycetech/applescript-core-apps1/refs/heads/main/scripts/install.sh | bash
```
